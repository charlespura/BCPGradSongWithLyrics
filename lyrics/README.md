# Lyrics folder

Put your lyrics text files here, one song per file.

This site loads lyrics from these paths:

- `lyrics/you-are-mine.txt`
- `lyrics/saranggola.txt`
- `lyrics/kaibigan-ko-paaralan-ko.txt`
- `lyrics/bcp-hymn.txt`
- `lyrics/we-made-it-through.txt`

Notes:

- Use plain text (`.txt`).
- Only add lyrics you have permission/rights to display.
